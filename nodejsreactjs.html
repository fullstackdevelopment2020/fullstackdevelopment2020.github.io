Using React JS.

  1. Download the 64-bit version of Node from here. Install Node. Don't forget to check a box telling you to "Automatically install the necessary tools ...". The first part of the installation takes a couple of minutes. The second part (the Chocolatey) takes around 10 minutes. If you get stuck, just close the PowerShell. It's going to be OK.
  2. On the command prompt type:
  3. node --version
  4. npm --version
  5. npm install -g create-react-app
  6. create-react-app --version
  7. create-react-app christmas2019---Note: It takes forever.
  8. cd christmas2019
  9. npm start --- It will start the web app.
  10. Use a different browser from your default one and type: http://localhost:3000/
  11. Click on the Learn React link. Good luck reading it.
  12. That's it. Just press CTRL C to stop the app.
