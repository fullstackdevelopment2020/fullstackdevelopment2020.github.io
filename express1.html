Working with ExpressJS.

Note: Use Visual Studio Code, which is highly recommended, or whatever text editor you want, like Brackets. You may use Visual Studio 2019, as well.
  1. Go to the command line and type:
    mkdir Express1
    cd Express1
    npm init --- We do this to create the package.json file. Accept all the defaults, but instead of index.js type app.js. For the rest of the prompts use whatever you want.
  2. type package.json --- We do this to read the contents of the package.json file.
  3. npm install express --- We do this to create the structure.
  4. tree --- We do this to see the file structure.
  5. tree /f --- We do this to see a more detailed view of the file structure.
  6. type nul > app.js--- We do this to create our entry point. Add this code to the app.js, save it. Close the file.
  7. node app.js --- We do this to run the application.
  8. Open Chrome or any web browser and type: http://localhost:3000 --- Don't close the browser yet.
  9. Open another command prompt and type:
    curl http://localhost:3000 --- We do this to confirm that the web server is Express. You can download curl from right here.
  10. C'est tout. That's it. Close whatever needs to be closed.
