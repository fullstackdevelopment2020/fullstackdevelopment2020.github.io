Hosting an ASP.NET Core app on Windows with IIS


  1. Install IIS on your computer. Just go to Control Panel (Programs, Windows Features On/Off). Don't forget to click on every single check box!
  2. Click the button below to download the magical component required to make this project work and install it: ASP.NET Core 3.0 Runtime (v3.0.1) - Windows Hosting Bundle Installer
  3. Restart IIS by typing iisreset on the command line. You may need to run it as an Administrator.
  4. Create an ASP.NET Core 3.0 MVC application. Name it AspNetCoreTest.
  5. Press F5 to build it and test it.
  6. Publish the application to a folder (just use default settings). Copy the publish folder to your C:\Inetpub\wwwroot folder and rename it to AspNetCoreTest.
  7. Open IIS (Administrative Tools) and create a new Application Pool (name it ForCoreApps). Make sure to set this: .NET CLR version: No Managed Code. Check the box to Start the Application Pool Immediately.
  8. Expand Sites -> Default Web Site. Convert the AspNetCoreTest folder to an application. Make sure to point to the ForCoreApps application pool.
  9. Right click on AspNetCoreTest -> Manage Application -> Browse
  10. Did it work?
